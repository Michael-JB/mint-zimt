## Setup

1. Click "Use this template" on GitHub to create your own private copy.
2. Create a Telegram bot with [@BotFather](https://t.me/botfather) and save the
   token as a repository secret named `TELEGRAM_BOT_TOKEN` (Settings -> Secrets
   and variables -> Actions).
3. Run the `Telegram chat ID` workflow (Actions tab) to get your chat ID and
   save it as a repository secret named `TELEGRAM_CHAT_ID`.
4. Get a Gemini API key from [Google AI
   Studio](https://aistudio.google.com/api-keys) and save it as a repository
   secret named `GEMINI_API_KEY`.

That's it.
